# 🚀 How to Use

## Option 1: Copy into Jupyter Notebook

1. Open Jupyter Notebook
2. Create a new notebook
3. Add a new cell
4. Change cell type to **Markdown**
5. Copy content from `markdown-guide.ipynb`
6. Paste into the Markdown cell
7. Press **Shift + Enter**

---

## Option 2: Use as Quick Reference

Simply open:

```text
README.md
```

or

```text
markdown-guide.md
```

and use it as your personal Markdown cheat sheet.

# 🧠 Jupyter Shortcuts

| Shortcut      | Action                   |
| ------------- | ------------------------ |
| M             | Convert to Markdown Cell |
| Y             | Convert to Code Cell     |
| Shift + Enter | Run Cell                 |
| A             | Insert Cell Above        |
| B             | Insert Cell Below        |
| DD            | Delete Cell              |
